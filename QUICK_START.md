## First Time Setup

1. **Double-click `INSTALL.bat`**

   - Checks if Python is installed
   - Installs all required libraries (Pillow, pynput, pygame)
   - Takes ~1 minute

2. **Double-click `RUN_APP.bat`**
   - Launches Crosshair Gambler Pro
   - Ready to use!

## Files

- **`INSTALL.bat`** - Installation script (run once)
- **`RUN_APP.bat`** - Launch application (use this every time)
